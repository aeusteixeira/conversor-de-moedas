# Conversor de Moedas

## Instalação

Para instalar as dependencias do projeto:

```sh
composer install
```

Em seguida, para abrir a aplicação local:
```sh
php artisan serve
```

Verifique a aplicação em seu navegador através do endereço:

```sh
127.0.0.1:8000
```

## Converções possíveis

| De | Para |
| ------ | ------ |
| BRL | EUR |
| BRL | USD |
| USD | BRL |
